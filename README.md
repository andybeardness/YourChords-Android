# About

An application with a collection of songs with chords in Russian and English

# Features

- **841 authors**
- **68885 songs**
- **Search by authors and songs**

# Dev Features

- **Kotlin**
- **Kotlin Coroutines**
- **Material3**
- **Material Icons**
- **Compose**
- **Compose Animations**
- **Compose Navigation**
- **Dynamic colors**
- **Hilt**
- **Room**
- **Ad**
- **Unit tests**
